# Design System (Claude/Anthropic-Inspired)

Full details in `DESIGN.md`. Key references below.

## Color Palette
- **Page background**: Parchment `#f5f4ed` (warm cream, never pure white)
- **Card surface**: Ivory `#faf9f5`
- **Primary text**: Near Black `#141413` (warm, olive-tinted)
- **Secondary text**: Olive Gray `#5e5d59`
- **Tertiary text**: Stone Gray `#87867f`
- **Brand accent**: Terracotta `#c96442` (CTAs only)
- **Borders (light)**: Border Cream `#f0eee6`
- **Dark surfaces**: `#30302e` containers, `#141413` backgrounds
- **ALL grays are warm-toned** — no cool blue-grays anywhere

## Typography
- **Headlines**: Playfair Display (serif), weight 500, line-height 1.10–1.30
- **Body/UI**: Nunito Sans, weight 400–500, line-height 1.60
- **Code**: monospace, only for actual code

## Key Rules
- Serif for headlines, sans-serif for UI — never mix
- Ring shadows (`0px 0px 0px 1px`) instead of drop shadows
- Border-radius: 8px standard, 12px buttons/inputs, 16–32px featured content
- Generous whitespace — editorial magazine pacing
- Light/dark section alternation for page rhythm
- Terracotta used sparingly — only primary CTAs

## Don'ts
- No cool blue-grays
- No bold (700+) on serif headlines
- No heavy drop shadows
- No pure white `#fff` as page background
- No sharp corners (< 6px) on buttons/cards
